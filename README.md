# Wiggle Font Showcase

This project is a single-page font showcase for the attached Wiggle typeface.

## Files

- `index.html` - page structure and content
- `style.css` - all visual styling
- `Wiggle-Regular.otf` / `Wiggle-Regular.ttf` - font files used by the page

## How to use

1. Open `index.html` in a browser.
2. Click inside the playground area and type your own text.
3. Keep the font files in the same folder so the stylesheet can load them.

## Notes

The page is built as a static HTML and CSS site, so no build step is required.
